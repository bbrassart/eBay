== README

eBay replica!

Built with Ruby on Rails.
You can create a new user, make a bid, post a product, make a comment, etc..
Images can be modified. You can add new ones, update, user profile, product thumbnail, etc...

Getting Started

Git clone
Bundle install
create database and seed it
You can log in with the following credentials:
Email: bart@gmail.com
PWD: bart

initialize the rails server
You are good to go
